# PrendeteRock

Proyecto Full-Stack Vue.js + C# para [descripción del proyecto]

## Estructura del Proyecto

- **frontend/** - Aplicación Vue.js
- **backend/** - API C# con .NET

## Requisitos

- Node.js 18+
- .NET 8+
- npm o yarn

## Instalación y Ejecución

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend/PrendeteRock.Api
dotnet restore
dotnet run
```
