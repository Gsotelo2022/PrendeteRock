# PrendeteRock Frontend

Aplicación Vue.js 3 con Vite

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
# La app estará disponible en http://localhost:3000
```

## Build para producción

```bash
npm run build
```

## Preview de producción

```bash
npm run preview
```

## Configuración

El archivo `vite.config.js` contiene la configuración del proxy para conectar con la API backend en `http://localhost:5000`.
